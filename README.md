# Flash File Manager

A lightweight Tkinter GUI application to automatically organize files in a target folder (default: Downloads) into categorized folders (Images, Documents, Videos, Audio, Archives, Software, Code, Design).

## Features

- Comprehensive file type categorization
- Option to separate into subfolders (category/subtype)
- Manual "CLEAN NOW" and background auto-agent (scans every 5 minutes)
- Built-in logging and statistics view

## Quick Start

1. Install Python 3.8+ (tkinter is included with standard CPython on Windows/Mac).
2. Run the app:

```bash
python flash_file.py
```

3. Use the GUI to select the target folder and press "CLEAN NOW" or enable the auto-agent.

## Notes
